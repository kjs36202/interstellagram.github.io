GitHub Pages 기반 블로그를 열었습니다.

이 저장소는 Jekyll 구조로 만들어져 있어서 `_posts` 폴더에 Markdown 파일만 추가하면 새 글이 자동으로 목록에 나타납니다.

글 파일 이름은 아래처럼 날짜를 포함하면 됩니다.

```text
YYYY-MM-DD-article-name.md
```

예시:

```text
2026-04-06-my-second-post.md
```

앞으로는 개발 기록, 트러블슈팅, 짧은 생각을 이곳에 정리해 둘 예정입니다.
